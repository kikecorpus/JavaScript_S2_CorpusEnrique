Reglas del Juego


  • Recibirás 5 cartas.
  • Haz clic en las cartas que deseas conservar.
  • Presiona "Repartir" para cambiar las demás cartas.
  • Se mostrará tu mano final.
  • Si tienes una buena combinación, ¡ganas!


Combinaciones Ganadoras:


  • Par de J o mejor - ejemplo: J❤️ + J♠️ + 8♠️ + 3♦ +K♦
  • Dos pares - ejemplo: 9❤️ + 9♠️ + 10♣ + 5♦ + 5❤️
  • Trío - ejemplo: 9❤️ + 9♣ + 9♦ + 5♣ + 10♣
  • Escalera - ejemplo: 1❤️ +2♣ +3♠️ +4♣ +5
  • Color - ejemplo: 4❤️ + A❤️ + 8❤️ + K❤️ + 9❤️
  • Full House - ejemplo: 9❤️ + 9♣ + 9♦ + J❤️ + J♠️
  • Póker - ejemplo: 9❤️ + 9♣ + 9♦ + 9♦ + 8♥
  • Escalera de color - ejemplo: 4♥ + 5♥ + 6♥ + 7♥ + 8♥
  • Escalera real - ejemplo: 10♥ + J♥ + Q♥ + K♥ + A♥
